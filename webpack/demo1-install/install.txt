介绍：Webpack 可以将多种静态资源 js、css、less 转换成一个静态文件，减少了页面的请求。

本地环境要求：node.js

安装建议安装到本地，项目之间互不影响

创建项目并完成安装
mkdir webpack-demo && cd webpack-demo
npm init -y
npm install webpack webpack-cli --save-dev

loader 类似Bable转译器

本demo使用命令
npm init -y
npm install webpack webpack-cli --save-dev
npm install --save lodash
npx webpack
